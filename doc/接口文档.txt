原生功能

1.上传图片接口【回调返回】(图片上传到阿里云，并返回上传好的图片地址)
	1.1从本地选择图片
	1.2从相机拍照
	1.3如果图片宽度大于800,则等比压缩到800再上传


	exec(success, error, "MyPlugin", "upImgMethod", [arg0]);

	arg0 = {url:"上传接口”, count: “允许上传图片的数量”, timestamp:”时间戳”, sign:”加密的字符串”} （比如这个格式请求：sign=1111&timestamp=1121212121）

	回调: （例如：）{code: 成功1，失败0, msg: 描述, imgUrl: [2017-03/12345.jpg,2017-03/12346.jpg]}

2.定位(Android需集成第三方,ios系统自带)【回调返回】
	exec(success, error, "MyPlugin", "positionMethod", [arg0]);

	arg0 : null

	回调: {		code: 成功1，失败0,
			msg: 描述,
			location_name:地址上方的名字
			location_adress:地址下方的名字
			location_lat: 纬度
			location_lng: 经度
			province:省（如果能取到）
			city：市（如果能取到）
			county：区、县（如果能取到）
			addr:详情地址（如果有则返回，没有就为空）}

3.分享【回调返回】
	exec(success, error, "MyPlugin", "shareUrlMethod", [arg0]);

	arg0 = {"pic":"图片地址绝对路径","title":"标题","desc":"描述","url":"实际跳转地址"}

	回调 : {code: 成功1，失败0, msg: 描述}

4.三方登录【回调返回】
	exec(success, error, "MyPlugin", "extLoginMethod", [arg0]);

	arg0 = {type: 种类（1微信，2QQ）}

	回调 : {	code: 成功1，失败0
		msg: 描述
		unionid:平台ID
		face:用户头像url
		nikename:昵称
		sex:性别（男，女，未知）
		city:城市（如果没有就为空）}
